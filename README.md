# Model viewer
Study project with OpenGL and Dear ImGUI.
Learning Source: https://learnopengl.com/

## Controls
W - Forward <br>
A - Left <br>
D - Right <br>
S - Back<br>
E - Up <br>
Q - Down <br>

Mouse Middle Button - View Rotation
